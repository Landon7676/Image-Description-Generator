# Image-Description-Generator

1. Activate the Enivronment: conda activate flickr

2. Run Caption Generator: python generate_caption.py --image path/to/image.jpg
example: python generate_caption.py --image data/1000092795.jpg

3. Expected output: 
Loading vocabulary...
Vocab size: 8642
Loading models...
Generating caption...

Image: data/flickr30k-images/flickr30k_images/1000092795.jpg
Caption: a man standing in a garden
